Hypertext Markup Language (HTML)

HTML Entities

While coding in html, you'll notice a lot of less than (<) and greater than (>) signs. However, if you would like to display those signs/special characters on your site, you would have to access them with &#entity_number; or &entity_name;. For example, less than sign has entity number 30 and entity name is "lt" (for less than).

Images (img tag)

  • Empty element that allows you to add images.
  • src = source attribute.
  • alt = specifies an alternate text for an image.
  • You can set width and height (if only sets one, the other is set to "auto").
  • Syntax: <img src="Images_folder/image.png" alt="description of image" width="x" height="y" >

Anchor Tag (A-tag)

  • Allows us to add links to our webpage.
  • href = Hypertext Reference.
  • When linking to other websites, you have to include the full url (including https:).
  • Target attribute is optional, by default target="_self" which opens in the same tab.
  • If you would like the link to open on a new tab (target="_blank").
  • Syntax: <a href="file.html"/"https:www.website.com" target="_blank"/"_self" > Name of link</a>

Cascading Style Sheets (CSS)

  • Not a programming language, it's a styling language used to modify the appearance of the content of web pages.
  • As HTML is used for the content, CSS is used for presentation of that content.
  • CSS has the ability to do things in many different ways

CSS Syntax:

selector {
<> property1: value;
<> property2: value;
}

}
